# Movie-Recommendation-System

Hello and welcome,
I have made a movie recommendation system that gets data insights based on the user's content and fetches five movies with similar content.
Users can search for the names of the movies in the search bar. A dropdown box is also provided for Convenience.
data set link: https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata?select=tmdb_5000_movies.csv
Tools used: 1. Pycharm  2.Jupyter notebook
Python Libraries: Numpy, pandas, pickle, and Kaggle API(open source) to fetch movie posters
