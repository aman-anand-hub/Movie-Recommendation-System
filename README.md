# Movie-Recommendation-System

Hello and welcome,
I have made a movie recommendation system which gets its data insights based on the content which user consumes and fetches five movies with similar content to the user.
User can search the names of the movies in the search bar. A dropdown box is also provided for Convenience.
data set link: https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata?select=tmdb_5000_movies.csv
